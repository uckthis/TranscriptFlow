🚀 Quick Start

  1. Load Media: Go to File > New Document with Media... or drag a video file directly into the player.
  2. Playback: Use the Tab key to toggle play/pause. Use Ctrl + Left/Right to skip through time.
  3. Transcription: Type in the editor. Use Ctrl + ; to insert a timestamp at the current playback position.
  4. Snapshots: Use Ctrl + 1-5 to quickly insert speaker names and timestamps (Snippets).

⏱️ Timecode Sync & Shifting

If you've started transcribing before realizing your media offset is wrong, you can fix it easily:

  • Transcript > Adjust Timecodes: Shift all timecodes in the document by a fixed amount (e.g., +1 second).
  • Transcript > Sync with Current Time: Align the transcript to the current video playback position. This is ideal if you're resuming work or handling multiple clips.

📋 Snippet Variables

Customize your snippets in Edit > Edit Snippets using these variables:

  • {$time} - Inserts the current media timecode.
  • {$media_name} - Inserts the filename of the loaded media.
  • {$media_path} - Inserts the full path of the media file.
  • {$time_offset(+00:00:01:00)} - Inserts a timecode with a specific offset.
